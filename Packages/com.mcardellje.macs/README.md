# Graph Keybinds:
- T - Start creating transition on selected / hovered node
- S - Create new empty state under mouse position
- C - Copy selected transitions
- V - Paste transitions starting from selected / hovered node

# Window Keybinds:
- Ctrl+L - Toggle inspector or project window lock
- Ctrl+W - Close current inspector window
- Ctrl+Shift+I - duplicate current inspector window, creating a new tab
- Shift+I - duplicate current inspector window, vertically split
- I - duplicate current inspector window, horizontally split
